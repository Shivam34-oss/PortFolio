Landing Page — Case Study

A fast, responsive marketing website built to increase conversions.

Problem

Client needed a lightweight landing page that loads fast on 2G/3G networks and converts visitors into leads.

Approach

  1. Mobile-first SCSS with critical CSS for hero
  2. Image optimization (WebP + responsive srcset)
  3. Simple contact form posting to serverless function

Tech Stack

HTML5 · SCSS · Vanilla JS · Netlify Functions

How to run locally

git clone ...
cd landing-page
npm install
npm run dev
